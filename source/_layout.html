Node ArcGIS

$ npm install arcgis

Client library for interfacing with ArcGIS, either a Server installation or ArcGIS Online. Works in server-side Node environments as well as the browser. Provides a set of convience methods, and structures the underlying REST API concepts and behaviors to align closer with the expectations of an API in the node environment.
